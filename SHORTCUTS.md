# QC Assistant Shortcuts

QC Assistant is optimized for fast inspection ballooning: keep your hand on the keyboard, stamp balloons on requirements, then edit details from the inspector or table.

## Tool Shortcuts

| Shortcut | Action |
|----------|--------|
| `B` | Balloon tool |
| `V` | Select tool |
| `H` | Pan tool |
| `T` | Text Select / OCR tool |
| `E` | Open edit/delete actions for the selected balloon |
| `Esc` | Close Help, close balloon actions, or cancel transient selection UI |

Shortcuts are ignored while typing in fields, tables, textareas, or dropdowns.

## Balloon Placement

1. Press `B` or choose the Balloon tool.
2. Click the dimension or note target once.
3. QC Assistant creates the next numbered balloon with a default leader line.
4. Repeat for every requirement.

The Balloon tool stays active after placement so you can stamp requirements quickly.

## Editing And Repositioning

- Single-click a balloon to select it.
- Drag the balloon circle to move the number.
- Drag the target handle to move the leader endpoint.
- Double-click a selected balloon, or press `E`, to open the floating edit/delete actions.
- Use the inspector or QC table for characteristic fields, tolerances, samples, and methods.

## Text And OCR Capture

1. Press `T` or choose Text Select.
2. Click selectable PDF text, or drag a box around raster text for OCR.
3. Send captured text to Drawing No, Rev, Supplier, Description, Requirement, Tolerance, or Notes.

## Export Workflow

1. Confirm every required balloon has a corresponding QC row.
2. Enter sample measurements.
3. Review status: incomplete rows stay `OPEN`; out-of-limit rows become `NG`.
4. Export the ballooned PDF and Excel QC/FAI workbook.
