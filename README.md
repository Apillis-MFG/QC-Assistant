# QC Assistant

Interactive drawing ballooning and QC/FAI report builder.

## Run Locally

```bash
pnpm install
pnpm dev
```

Open `http://127.0.0.1:5173/`.

## Build

```bash
pnpm build
```

## Deploy To Vercel

Use `QC-Assistant` as the Vercel project root.

- Framework: Vite
- Build command: `pnpm build`
- Output directory: `dist`
